## ◆開発の背景
以前授業で作成したガチャガチャ巡りの紹介サイトは、処理が重く快適に閲覧できませんでした。
そこで、興味があったNext.jsを使用して、より快適に閲覧できるように改善しようと思い、開発しました。<br><br>


## ◆使用技術
